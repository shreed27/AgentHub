# Trading Orchestrator Frontend

A high-performance, aesthetic frontend for the Autonomous Trading Platform.

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features

- **Glassmorphism UI**: Premium dark mode design with blurred backgrounds.
- **Micro-interactions**: Smooth hover states and entrance animations.
- **Responsive**: Fully responsive layout for all devices.
